6. Emacs

Emacsen 21 and newer have partial euro support. The following elisp should work: 

 (set-face-font
      'default '"-*-courier-medium-r-*-*-*-120-*-*-*-*-iso8859-15")

Note that you cannot write Euro characters. You can only see them.
