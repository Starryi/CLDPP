4. SuSE ja UTF-8

SuSE siirtyi UTF-8 oletukseen versiossa 9.1. Jos sen haluaa vaihtaa niin sen voi tehdä YaST:illa seuraavasti: Yast2 -> System -> Sysconfig editor -> Search -> RC_LANG.
