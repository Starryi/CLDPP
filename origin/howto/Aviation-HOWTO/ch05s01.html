1. PilotPal

PilotPal is a Java applet designed to assist (student) pilots studying for the US FAA written examinations. The latest version provides study questions for both the Private Pilot and Instrument examinations. If you don't have a high bandwidth connection to the Internet, you may want to download the applet and install it on your own machine.
