2. Gather the tools

We will need a few pieces of software. Get the following packages, and install them where specified.

2.1. For the Server:

  • pppd (version 2.3 or greater)

  • ssh (version 1.2.26 or better)

2.2. For the Client:

  • pppd (same version as server)

  • ssh

  • pty-redir
