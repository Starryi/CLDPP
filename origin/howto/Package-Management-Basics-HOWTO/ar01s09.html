Conclusion and Further Reading

This guide provides an overview of basic operations that can be cross-referenced between systems, but only scratches the surface of a complex topic. For greater detail on a given system, you can consult the following resources:

  1. This guide covers Ubuntu and Debian package management in detail.
  2. There’s an official CentOS guide to managing software with yum.
  3. There’s a Fedora wiki page about dnf, and an official manual for dnf itself
  4. This guide covers FreeBSD package management using pkg.
  5. The FreeBSD Handbook contains a section on using the Ports Collection.
  6. OpenSuSE documentation for Zypper and YaST.
