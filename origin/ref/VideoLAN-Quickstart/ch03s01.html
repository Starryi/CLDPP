1.  Unicast an MPEG-1, MPEG-2, or MPEG-4 / DivX file

Start VLC on the server 

% vlc -vvv video1.xyz --sout udp:client.example.org

where video1.xyz is the file you want to stream and client.example.org is the DNS name of the client you want to stream to (you can put an IP address instead).

Start VLC on the client 

% vlc -vvv udp:
