14. Aviation

Many people are using laptops for aviation related topics. The Aviation HOWTO provides pointers to software packages that run under the Linux operating system and are useful to private, commercial, or military pilots. The ultimate goal is to enable pilots to use the Linux operating system for all their aviation related computing needs.
